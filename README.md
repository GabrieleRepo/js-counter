## About The Project

![Product Name Screen Shot][product-screenshot]
ðŸ”— [Live Demo](https://gz-counter.netlify.app)

Interactive Counter is a simple web application developed using **JavaScript**.

The user can:
- Increase the counter value
- Decrease the counter value
- Reset the counter to zero
- Change the step value dynamically using a delta control

At the start of the application, the user is asked to enter a **positive number**
that will be used as the step value for the counter.

The entire interface is dynamically generated using JavaScript and styled with Bootstrap 5.

## Usage

1. Visit the live demo: [gz-counter.netlify.app](https://gz-counter.netlify.app)
2. A welcome message will be displayed
3. Enter a positive number for the counter step
4. Use the buttons to increase, decrease or reset the counter
5. Adjust the step value dynamically using the delta control
